# ARM Tester

ARM Tester is a tool to estimate CPU performence for ARM devices.

## Installation

Use [git](https://github.com/Ech4le/ARM_Tester/) to install ARM Tester.

```bash
git clone https://github.com/Ech4le/ARM_Tester/
```

## Usage

```bash
python3 main.py
```
